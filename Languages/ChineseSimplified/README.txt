# 更强的屋顶 [![RimWorld Alpha 17](https://img.shields.io/badge/RimWorld-Alpha%2017-brightgreen.svg)](http://rimworldgame.com/) 

增加可定制屋顶，透明温室屋顶和光伏太阳能屋顶。根据Vendan's的灵感重制。

## MON描述
- 透明的屋顶可以让光线毫无保留的照进来
- 太阳能屋顶和控制器可以使用更环保的能源
- 增加移除厚重的屋顶
- 增加厚石屋顶
- 研究中新增加四个新技术选项

## 太阳能屋顶和控制器描述
- 每个太阳能控制器最大的功率是2500W
- 每个太阳能屋顶瓦在阳光充足的环境下产生200W电力
- 连接到太阳能屋顶瓦的控制器数量没有限制。

## 注解
- 新的科技研究项目位于“研究”栏中
- 关于屋顶是怎么使用的问题，最简单的就是将屋顶选项默认为新屋顶
- 一个单独的太阳能控制器只能连接一段太阳能屋顶
- 每个太阳能电池板产生50w（也许不是？）
